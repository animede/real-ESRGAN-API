# real-ESRGAN-API
Real-ESRGAN real-time API serevr

## 環境構築

tkh作成後　リポジトリから　requirements.txt　をコピーします
```
python3.11 -m venv tkh
source tkh/bin/activate
cd tkh
pip install -r requirements.txt
```


このリポジトリは以下を引き継いでいます。
